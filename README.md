# UAVs
Documentation of Class 1 UAVs and Specs


The U.S. department of defense (DOD) categorizeses UAS in 5 groups in accordance to certain attributes--the aircraft's maximum gross takeoff weight, operating altitude, and speed. 

Group/Class 1 drones are under the scope of this project. 

Maximum Gross Takeoff Weight (lbs) : 0-20
Normal Operating Altitude (ft): < 1200 AGL ( height of Class G airspace ) 
Speed (KIAS) :  100 kts (knots)
